### 防抖
防抖函数：当事件在n秒内连续被触发，只执行最后一次

节流函数：当事件连续被触发时，每隔n秒执行一次

看一个例子

```js
var timer = null
function debounce (func, wait) {
    // timer定时器
    // var timer = null
    return function () {
        if (timer !== null) {
            clearTimeout(timer)
        }
        timer = setTimeout(func, wait)
    }
}
```
